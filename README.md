# Statistikaamet Quiz App

## Project Overview

This project is an interactive quiz application built with React. The application presents users with multiple-choice questions and provides immediate feedback on their answers. At the end of the quiz, the user is shown a summary of their results, including their selected answers, whether they were correct or incorrect, and the final score.

---

## Technologies

- React
- Vite
- TypeScript
- CSS
- Playwright

---

## Running the Application

### 1. Clone the repository

```bash
git clone https://github.com/agnessatnd/statistikaamet-quiz-app.git
```

### 2. Navigate to the project directory

```bash
cd statistikaamet-quiz-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will be available at:

http://localhost:5173

---

## Running Tests

The project includes Playwright end-to-end tests.

To run the tests:

```bash
npx playwright test
```
